## Examples

### Size

Use the `size` prop to control the size of the link. The prop also provides correct line height and corrective letter spacing—as text size increases, the relative line height and letter spacing decrease.

```jsx live=true scroll=true
<Flex direction="column" gap="3">
  <Link size="1">Sign up</Link>
  <Link size="2">Sign up</Link>
  <Link size="3">Sign up</Link>
  <Link size="4">Sign up</Link>
  <Link size="5">Sign up</Link>
  <Link size="6">Sign up</Link>
  <Link size="7">Sign up</Link>
  <Link size="8">Sign up</Link>
  <Link size="9">Sign up</Link>
</Flex>
```

### Weight

Use the `weight` prop to set the text weight.

```jsx live="true"
<Flex direction="column">
  <Link weight="regular">Sign up</Link>
  <Link weight="medium">Sign up</Link>
  <Link weight="bold">Sign up</Link>
</Flex>
```

### Color

Use the `color` prop to assign a specific [color](/themes/docs/theme/color).

```jsx live=true scroll=true
<Flex direction="column">
  <Link color="indigo">Sign up</Link>
  <Link color="cyan">Sign up</Link>
  <Link color="orange">Sign up</Link>
  <Link color="crimson">Sign up</Link>
</Flex>
```

### High-contrast

Use the `highContrast` prop to increase color contrast with the background.

```jsx live=true scroll=true
<Flex direction="column">
  <Link color="gray">Sign up</Link>
  <Link color="gray" highContrast>
    Sign up
  </Link>
</Flex>
```

### Underline

Use the `underline` prop to manage the visibility of the underline affordance.

```jsx live="true" scroll=true
<Flex direction="column">
  <Link underline="hover">Sign up</Link>
  <Link underline="always">Sign up</Link>
</Flex>
```
