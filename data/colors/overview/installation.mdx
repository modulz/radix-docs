# Installation

<Description>Install Radix Colors.</Description>

Radix Colors is available as an npm package.

## npm

To install and save Radix Colors in your `package.json` dependencies, run:

```jsx
// with npm
npm install @radix-ui/colors

// with yarn
yarn add @radix-ui/colors
```
