# Installation

<Description>How to install Radix Colors.</Description>

Radix Colors is available as an [npm package](https://www.npmjs.com/package/@radix-ui/colors).

## npm

If you're using CSS-in-JS, install the npm package.

```jsx
// with npm
npm install @radix-ui/colors

// with yarn
yarn add @radix-ui/colors
```
