# Color contrast tests

<Description>
  A test suite for testing color contrast in accordance with WCAG
  guidelines.
</Description>

## AA Normal Text

The `900` step is designed for 400 weight text in any size, and guaranteed to work well on a `200` background. Each test should pass 4.5:1 contrast ratio.

<ColorTestText />

## AA Large Text

The `800` step is designed for solid backgrounds, and guaranteed to work well behind white text in 600 weight at 19px. Each test should pass 3:1 contrast ratio.

<ColorTestTextLarge />

These bright scales are designed for black foreground text.
