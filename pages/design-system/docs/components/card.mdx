#### Simple Card

```jsx live
<Box css={{ maxWidth: 250 }}>
  <Card css={{ p: '$3' }}>
    <Subheading css={{ mb: '$2' }}>
      Modulz raises $4.2M to close the gap between design and code
    </Subheading>
    <Text size="3" color="gray" css={{ lineHeight: '23px' }}>
      Modulz is a visual code editor that empowers teams to design, develop, document and deploy a design system, without writing code.
    </Text>
  </Card>
</Box>
```

#### Adding content
You can add almost anything you want inside a `Card`.

```jsx live
<Box css={{ maxWidth: 250 }}>
  <Card css={{ p: '$3' }}>
    <Image src="https://images.unsplash.com/photo-1453235421161-e41b42ebba05?ixlib=rb-1.2.1&ixid=MXwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHw%3D&auto=format&fit=crop&w=2550&q=80" css={{ mb: '$3' }} />
    <Subheading css={{ mb: '$2' }}>
      Modulz raises $4.2M to close the gap between design and code
    </Subheading>
    <Text size="3" color="gray" css={{ lineHeight: '23px' }}>
      Modulz is a visual code editor that empowers teams to design, develop, document and deploy a design system, without writing code.
    </Text>
  </Card>
</Box>
```
