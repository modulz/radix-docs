features:
  - Quickly make any icon accessible by wrapping it and providing a meaningful label.
  - No visual difference, but announced correctly by screen readers.
---

## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-accessible-icon
```

## Anatomy

Import the component.

```jsx
import { AccessibleIcon } from '@radix-ui/react-accessible-icon';

export default () => <AccessibleIcon />;
```

## Basic example

Create your accessible icon component from the primitive.

```jsx live manual
import * as AccessibleIconPrimitive from '@radix-ui/react-accessible-icon';
import { CrossCircledIcon } from '@modulz/radix-icons';

export default () => (
  <AccessibleIconPrimitive.Root label="Close">
    <CrossCircledIcon />
  </AccessibleIconPrimitive.Root>
);
```
