title: Context Menu
name: context-menu
version: 0.0.1-1
description: Useful for displaying a list of contextual actions by right clicking.
features:
  - Supports items, labels, groups of items.
  - Supports checkable items (single or multiple).
  - Customise side, alignment, offsets, collision handling.
  - Focus is fully managed.
  - Full keyboard navigation.
  - Dismissing and layering behavior is highly customisable.

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-context-menu
```

### Assembly

Import the component, add style to each part and piece the parts together.

```jsx
import {
  ContextMenu,
  Trigger,
  MenuPopper,
  MenuItem,
  MenuSeparator,
} from '@radix-ui/react-context-menu';

export default () => (
  <ContextMenu>
    <Trigger>Trigger</Trigger>
    <MenuPopper>
      <MenuItem>Item</MenuItem>
      <MenuSeparator />
      <MenuItem>Item</MenuItem>
    </MenuPopper>
  </ContextMenu>
);
```

### Basic example

Here's a basic example with styled parts.

```jsx preview manual
import React from 'react';
import * as ContextMenuPrimitive from '@radix-ui/react-context-menu';
import { styled } from './stitches.config';

const MenuPopper = styled(ContextMenuPrimitive.Popper, {
  // separator styles
});

const MenuItem = styled(ContextMenuPrimitive.MenuItem, {
  // ...item styles
});

const MenuSeparator = styled(ContextMenuPrimitive.MenuSeparator, {
  // ...separator styles
});

export default () => (
  <ContextMenuPrimitive.Root>
    <ContextMenuPrimitive.Trigger>
      <TriggerArea />
    </ContextMenuPrimitive.Trigger>
    <MenuPopper sideOffset={-5}>
      <MenuItem>Undo</MenuItem>
      <MenuItem>Redo</MenuItem>
      <MenuSeparator />
      <MenuItem>Cut</MenuItem>
      <MenuItem>Copy</MenuItem>
      <MenuItem>Paste</MenuItem>
    </MenuPopper>
  </ContextMenuPrimitive.Root>
);
```

### API Reference

#### ContextMenu

`ContextMenu` is the root component.

#### ContextMenuTrigger

`ContextMenuTrigger` is the area that triggers the `ContextMenu`. Wrap this around the thing you want your `ContextMenu` to open when right-clicking (or using the relevant keyboard shortcuts).

#### ContextMenuPopper

`ContextMenuPopper` is the component that pops out in an open `ContextMenu`.

#### ContextMenuGroup

Used to group multiple `ContextMenuItem`.

#### ContextMenuLabel

Uused to render a label. It won't be focusable using arrow keys.

Renders when the parent `ContextMenuCheckboxItem` or `ContextMenuRadioItem` is checked. You can style this element directly, or you can use it as a wrapper to put an icon into, or both.

### Styling

Add your own styles to each part. Below are some basic recommendations.

```js

  cursor: 'default',
  '&[data-disabled]': {
		color: 'slategray'
	},

  height: 1,
  backgroundColor: 'gainsboro',
});
```
