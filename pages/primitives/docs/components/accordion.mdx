  - Full keyboard navigation.
  - Can be controlled or uncontrolled.

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-accordion
```

Then import the component in your component file.

```jsx
// Accordion.tsx
import * as AccordionPrimitive from '@interop-ui/react-accordion';
```

### Basic example

Create your styled slider component from the slider primitives.

```jsx live manual
// Slider.tsx
// import * as AccordionPrimitive from '@interop-ui/react-accordion';

const AccordionItem = styled(AccordionPrimitive.Item, {
  borderTop: '1px solid $gray500',

  '&:last-of-type': {
    borderBottom: '1px solid $gray500',
  },
});

const AccordionHeader = styled(AccordionPrimitive.Header, {
  margin: 0,
  display: 'flex',
});

const AccordionButton = styled(AccordionPrimitive.Button, {
  backgroundColor: 'transparent',
  border: 'none',
  padding: 10,
  flex: 1,
});

const AccordionPanel = styled(AccordionPrimitive.Panel, {
  padding: 10,
});

render(
  <AccordionPrimitive>
    <AccordionItem value="item-1">
      <AccordionHeader>
        <AccordionButton>Item 1</AccordionButton>
      </AccordionHeader>
      <AccordionPanel>
        <div>Here goes the content for the accordion item 1.</div>
      </AccordionPanel>
    </AccordionItem>

    <AccordionItem value="item-2">
      <AccordionHeader>
        <AccordionButton>Item 2</AccordionButton>
      </AccordionHeader>
      <AccordionPanel>
        <div>Here goes the content for the accordion item 2.</div>
      </AccordionPanel>
    </AccordionItem>

    <AccordionItem value="item-3">
      <AccordionHeader>
        <AccordionButton>Item 3</AccordionButton>
      </AccordionHeader>
      <AccordionPanel>
        <div>Here goes the content for the accordion item 3.</div>
      </AccordionPanel>
    </AccordionItem>
  </AccordionPrimitive>
);
```

#### AccordionItem

`AccordionItem` contains all of the parts of a collapsible section inside of an `Accordion`.

#### AccordionHeader

`AccordionHeader` contains the content for the parts of an `AccordionItem` that will be visible whether or not its content is collapsed.

#### AccordionButton

`AccordionButton` is the trigger that toggles the collapsed state of an `AccordionItem`. It should always be nested inside of an `AccordionHeader`.

#### AccordionPanel

`AccordionPanel` contains the collapsible content for an `AccordionItem`.
