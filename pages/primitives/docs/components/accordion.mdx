---
title: Accordion
name: accordion
version: 0.0.1-13
description: A vertically stacked set of interactive headings that each reveal an associated section of content.
aria: https://www.w3.org/TR/wai-aria-practices-1.2/#accordion
features:
  - Full keyboard navigation.
  - Expands one panel at a time.
  - Can be controlled or uncontrolled.
---

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-accordion
```

### Anatomy

Import the components and piece the parts together.

```jsx
import {
  Root,
  Item,
  Header,
  Button,
  Panel,
} from '@radix-ui/react-accordion';

export default () => (
  <Root>
    <Item>
      <Header>
        <Button />
      </Header>
      <Panel />
    </Item>
  </Root>
);
```

### Basic example

Create your styled accordion component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as AccordionPrimitive from '@radix-ui/react-accordion';

const Accordion = styled(AccordionPrimitive.Root, {
  borderTop: '1px solid gainsboro',
});

const Item = styled(AccordionPrimitive.Item, {
  borderBottom: '1px solid gainsboro',
});

const Header = styled(AccordionPrimitive.Header, {
  margin: 0,
  display: 'flex',
});

const Button = styled(AccordionPrimitive.Button, {
  backgroundColor: 'transparent',
  border: 'none',
  padding: 10,
  flex: 1,
});

const Panel = styled(AccordionPrimitive.Panel, {
  padding: 10,
});

export default () => (
  <Accordion>
    <Item value="item-1">
      <Header>
        <Button>Item 1</Button>
      </Header>
      <Panel>Here goes the content for the accordion item 1.</Panel>
    </Item>

    <Item value="item-2">
      <Header>
        <Button>Item 2</Button>
      </Header>
      <Panel>Here goes the content for the accordion item 2.</Panel>
    </Item>

    <Item value="item-3">
      <Header>
        <Button>Item 3</Button>
      </Header>
      <Panel>Here goes the content for the accordion item 3.</Panel>
    </Item>
  </Accordion>
);
```

#### Header

`Header` wraps a `button` and will be an `h3` by default. Use the `as` prop to update it to the appropriate heading level for your page.

#### Button

`Button` toggles the collapsed state of an `Item`. It should always be nested inside of an `Header`.

#### Panel

`Panel` contains the collapsible content for an `Item`.

### Examples

#### With directional Icon

```jsx live manual compact
// import { styled } from 'path-to/stitches.config';
import * as AccordionPrimitive from '@radix-ui/react-accordion';
import { ChevronDownIcon } from '@modulz/radix-icons';

const Accordion = styled(AccordionPrimitive.Root, {
  borderTop: '1px solid gainsboro',
});

const Item = styled(AccordionPrimitive.Item, {
  borderBottom: '1px solid gainsboro',
});

const Header = styled(AccordionPrimitive.Header, {
  margin: 0,
  display: 'flex',
});

const Button = styled(AccordionPrimitive.Button, {
  backgroundColor: 'transparent',
  border: 'none',
  padding: 10,
  flex: 1,
  display: 'flex',
  alignItems: 'center',
  justifyContent: 'space-between',
});

const Panel = styled(AccordionPrimitive.Panel, {
  padding: 10,
});

const AccordionChevron = styled(ChevronDownIcon, {
  transition: 'transform 300ms',

  '[data-state=open] &': {
    transform: 'rotate(180deg)',
  },
});

export default () => (
  <AccordionPrimitive.Root>
    <Item value="item-1">
      <Header>
        <Button>
          Item 1
          <AccordionChevron aria-hidden />
        </Button>
      </Header>
      <Panel>Here goes the content for the accordion item 1.</Panel>
    </Item>

    <Item value="item-2">
      <Header>
        <Button>
          Item 2
          <AccordionChevron aria-hidden />
        </Button>
      </Header>
      <Panel>Here goes the content for the accordion item 2.</Panel>
    </Item>

    <Item value="item-3">
      <Header>
        <Button>
          Item 3
          <AccordionChevron aria-hidden />
        </Button>
      </Header>
      <Panel>Here goes the content for the accordion item 3.</Panel>
    </Item>
  </AccordionPrimitive.Root>
);
```
