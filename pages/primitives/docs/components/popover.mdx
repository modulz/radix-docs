---
title: Popover
name: popover
version: 0.0.1-13
description: Displays rich content in a portal, triggered by a button.
aria: https://www.w3.org/TR/wai-aria-practices-1.2/#dialog_modal
features:
  - Can be controlled or uncontrolled.
  - Customise side, alignment, offsets, collision handling.
  - Optionally render a pointing arrow.
  - Focus is fully managed and customisable.
  - Dismissing and layering behavior is highly customisable.
---

## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-popover
```

## Anatomy

Import the components and piece the parts together.

```jsx
import {
  Popover,
  Trigger,
  Content,
  Close,
  Arrow,
} from '@radix-ui/react-popover';

export default () => (
  <Popover>
    <Trigger />
    <Content>
      <Close />
      <Arrow />
    </Content>
  </Popover>
);
```

## Basic example

Create your styled popover component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as PopoverPrimitive from '@radix-ui/react-popover';

const Content = styled(PopoverPrimitive.Content, {
  borderRadius: 3,
  padding: '20px',
  fontSize: 14,
  backgroundColor: 'gainsboro',
  color: 'black',
});

const Arrow = styled(PopoverPrimitive.Arrow, {
  fill: 'gainsboro',
});

export default () => (
  <PopoverPrimitive.Root>
    <PopoverPrimitive.Trigger>Trigger</PopoverPrimitive.Trigger>
    <Content>
      <h3>Popover content</h3>
      <p>Are you sure you wanna do this?</p>
      <PopoverPrimitive.Close>Yes</PopoverPrimitive.Close>
      <Arrow />
    </Content>
  </PopoverPrimitive.Root>
);
```

### Trigger

`PopoverTrigger` is the button that triggers the `Popover`. It is also used as the anchor the `PopoverContent` is positioned against.
