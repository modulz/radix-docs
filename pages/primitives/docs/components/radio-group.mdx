### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-radio-group
```

### Anatomy

Import the components and piece the parts together.

```jsx
import {
  RadioGroup,
  Item,
  Indicator,
} from '@radix-ui/react-radio-group';

export default () => (
  <RadioGroup>
    <Item>
      <Indicator />
    </Item>
  </RadioGroup>
);
```

### Basic example

Create your styled radio group component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as RadioGroupPrimitive from '@radix-ui/react-radio-group';

const Radio = styled(RadioGroupPrimitive.Item, {
  appearance: 'none',
  backgroundColor: 'transparent',
  border: 'none',
  padding: 0,
  borderRadius: '50%',
  boxShadow: 'inset 0 0 0 1px gainsboro',
  width: 15,
  height: 15,
  display: 'inline-flex',
  alignItems: 'center',
  justifyContent: 'center',
  verticalAlign: 'middle',

  '& ~ &': { marginLeft: 5 },

  '&:focus': {
    outline: 'none',
    boxShadow: 'inset 0 0 0 1px dodgerblue, 0 0 0 1px dodgerblue',
  },
});

const Indicator = styled(RadioGroupPrimitive.Indicator, {
  width: 7,
  height: 7,
  borderRadius: '50%',
  backgroundColor: 'dodgerblue',
});

export default () => (
  <RadioGroupPrimitive.Root>
    <Radio value="cat">
      <Indicator />
    </Radio>
    <Radio value="dog">
      <Indicator />
    </Radio>
    <Radio value="rabbit">
      <Indicator />
    </Radio>
  </RadioGroupPrimitive.Root>
);
```

### Accessibility

Adheres to the [Radio Group WAI-ARIA design pattern](https://www.w3.org/TR/wai-aria-practices/#radiobutton) and uses [roving tabindex](https://www.w3.org/TR/wai-aria-practices/examples/radio/radio-1/radio-1.html) to manage focus movement among radio items.
