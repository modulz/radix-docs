### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-announce
```

Then import the component in your component file.

```jsx
// Announce.tsx
import * as AnnouncePrimitive from '@interop-ui/react-announce';
```

### Basic example

```jsx
// Announce.tsx
// import * as AnnouncePrimitive from '@interop-ui/react-announce';

render(<AnnouncePrimitive>Welcome to Radix UI.</AnnouncePrimitive>);
```
