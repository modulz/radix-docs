---
title: Tooltip
name: tooltip
version: 0.0.1-13
description: A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.
aria: https://www.w3.org/TR/wai-aria-practices-1.2/#tooltip
features:
  - Built-in time machine to control display delay.
  - Full keyboard support.
  - Opens when the trigger is focused.
  - Closes when the trigger is clicked.
---

## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-tooltip
```

## Anatomy

Import the components and piece the parts together.

```jsx
import {
  Tooltip,
  Trigger,
  Content,
  Arrow,
} from '@radix-ui/react-switch';

export default () => (
  <Tooltip>
    <Trigger />
    <Content />
    <Arrow />
  </Tooltip>
);
```

## Basic example

Create your styled tooltip component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as TooltipPrimitive from '@radix-ui/react-tooltip';
import { BlendingModeIcon } from '@modulz/radix-icons';

const TooltipContent = styled(TooltipPrimitive.Content, {
  borderRadius: 3,
  padding: '5px 10px',
  fontSize: 14,
  backgroundColor: 'gainsboro',
  color: 'black',
});

const Arrow = styled(TooltipPrimitive.Arrow, {
  fill: 'gainsboro',
});

export default () => (
  <TooltipPrimitive.Root>
    <TooltipPrimitive.Trigger>
      <BlendingModeIcon aria-label="Blending mode" />
    </TooltipPrimitive.Trigger>
    <TooltipContent>
      Blend mode
      <Arrow />
    </TooltipContent>
  </TooltipPrimitive.Root>
);
```

### Trigger

The button that triggers the `Tooltip`. It is also used as the anchor the `TooltipContent` is positioned against.
