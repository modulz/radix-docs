### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-tooltip
```

Then import the component in your component file.

```jsx
// Tooltip.tsx
import * as TooltipPrimitive from '@radix-ui/react-tooltip';
```

### Basic example

```jsx live manual
// Tooltip.tsx
// import * as TooltipPrimitive from '@radix-ui/react-tooltip';
// import { IconButton } from '@radix-ui/ds-button'
// import { BlendingModeIcon } from '@modulz/radix-icons'

const TooltipContent = styled(TooltipPrimitive.Content, {
  borderRadius: 3,
  padding: '5px 10px',
  fontSize: 14,
  backgroundColor: 'gainsboro',
  color: 'black',
});

render(
  <TooltipPrimitive>
    <TooltipPrimitive.Trigger as={IconButton}>
      <BlendingModeIcon aria-label="Blending mode" />
    </TooltipPrimitive.Trigger>
    <TooltipPrimitive.Popper>
      <TooltipContent>Blending mode</TooltipContent>
      <TooltipPrimitive.Arrow style={{ fill: 'gainsboro' }} />
    </TooltipPrimitive.Popper>
  </TooltipPrimitive>
);
```

#### TooltipTrigger

`TooltipTrigger` is the button that triggers the `Tooltip`. It is also used as the anchor the `TooltipPopper` is positioned against.

#### TooltipArrow

`TooltipArrow` is the optional arrow element to render for the `Tooltip`. This can be used to help visually link the `TooltipTrigger` with the `TooltipPopper`. Must live inside `TooltipPopper`.
