## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-separator
```

## Anatomy

Import the components and piece the parts together.

```jsx
import { Separator } from '@radix-ui/react-separator';

export default () => <Separator />;
```

## Basic example

Create your styled separator component from the primitive part.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as SeparatorPrimitive from '@radix-ui/react-separator';

const Separator = styled(SeparatorPrimitive.Root, {
  backgroundColor: 'gainsboro',
  height: 1,
  margin: '20px 0',
});

export default () => (
  <>
    Content
    <Separator />
    Content
  </>
);
```
