## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-tabs
```

## Anatomy

Import the components and piece the parts together.

```jsx
import { Tabs, List, Tab, Panel } from '@radix-ui/react-tabs';

export default () => (
  <Tabs>
    <List>
      <Tab />
    </List>
    <Panel />
  </Tabs>
);
```

## Basic example

Create your styled tabs component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as TabsPrimitive from '@radix-ui/react-tabs';

const Tabs = styled(TabsPrimitive.Root, {
  display: 'flex',
  flexDirection: 'column',
});

const List = styled(TabsPrimitive.List, {
  flexShrink: 0,
  display: 'flex',
  borderBottom: '1px solid gainsboro',
});

const Tab = styled(TabsPrimitive.Tab, {
  flexShrink: 0,
  padding: '10px 20px',
  color: 'slategray',
  userSelect: 'none',

  '&:hover': { color: 'black' },

  '&[data-state="active"]': {
    color: 'black',
    boxShadow:
      'inset 0 -1px 0 0 currentColor, 0 1px 0 0 currentColor',
  },
});

const Panel = styled(TabsPrimitive.Panel, {
  flexGrow: 1,
  padding: 20,
});

export default () => (
  <Tabs defaultValue="tab1">
    <List aria-label="tabs example">
      <Tab value="tab1">One</Tab>
      <Tab value="tab2">Two</Tab>
      <Tab value="tab3">Three</Tab>
    </List>
    <Panel value="tab1">Tab one content</Panel>
    <Panel value="tab2">Tab two content</Panel>
    <Panel value="tab3">Tab three content</Panel>
  </Tabs>
);
```

### List

`TabsList` displays the `TabsTab` components aligned along the edge of the active `TabsPanel`.
