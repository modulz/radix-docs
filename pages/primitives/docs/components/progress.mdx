## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-progress
```

### Anatomy

Import the components and piece the parts together.

```jsx
import { Progress, Indicator } from '@radix-ui/react-progress';

export default () => (
  <Progress>
    <Indicator />
  </Progress>
);
```

## Basic example

Create your styled progress component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as ProgressBarPrimitive from '@radix-ui/react-progress';

const Progress = styled(ProgressBarPrimitive.Root, {
  position: 'relative',
  height: 10,
  overflow: 'hidden',
  borderRadius: 5,
  background: 'gainsboro',
});

const ProgressIndicator = styled(ProgressBarPrimitive.Indicator, {
  boxSizing: 'border-box',
  position: 'absolute',
  backgroundColor: 'dodgerblue',
  height: '100%',
});

export () => (
  <Progress value={50}>
    <ProgressIndicator style={{ width: '50%' }} />
  </Progress>
);
```

## Accessibility

Adheres to the [`progressbar` role requirements](https://www.w3.org/TR/wai-aria-1.2/#progressbar).
