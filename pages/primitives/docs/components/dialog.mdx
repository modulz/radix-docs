## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-dialog
```

## Anatomy

Import the components and piece the parts together.

```jsx
import {
  Root,
  Overlay,
  Content,
  Close,
} from '@radix-ui/react-dialog';

export default () => (
  <Root>
    <Trigger />
    <Overlay />
    <Content>
      <Close />
    </Content>
  </Root>
);
```

## Basic example

Create your styled dialog component from the dialog primitives.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as DialogPrimitive from '@radix-ui/react-dialog';

const Overlay = styled(DialogPrimitive.Overlay, {
  backgroundColor: 'rgba(0, 0, 0, .15)',
  position: 'fixed',
  top: 0,
  right: 0,
  bottom: 0,
  left: 0,
});

const Content = styled(DialogPrimitive.Content, {
  position: 'fixed',
  top: '50%',
  left: '50%',
  transform: 'translate(-50%, -50%)',
  minWidth: 200,
  maxWidth: 'fit-content',
  maxHeight: '85vh',
  padding: 20,
  marginTop: '-5vh',
  backgroundColor: 'white',
  borderRadius: 6,

  '&:focus': {
    outline: 'none',
  },
});

export default () => (
  <DialogPrimitive.Root>
    <DialogPrimitive.Trigger>Open</DialogPrimitive.Trigger>
    <Overlay />
    <Content>
      <p>Dialog content</p>
      <DialogPrimitive.Close>Close</DialogPrimitive.Close>
    </Content>
  </DialogPrimitive.Root>
);
```

### Trigger

The button that triggers the `Dialog`.

### Overlay
