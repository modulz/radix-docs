### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-alert-dialog
```

Then import the component in your component file.

```jsx
// AlertDialog.tsx
import * as AlertDialogPrimitive from '@radix-ui/react-alert-dialog';
```

### Basic example

```jsx live manual
// Dialog.tsx
// import * as AlertDialogPrimitive from '@radix-ui/react-dialog';
// import { Button } from '@radix-ui/ds-button'

const Overlay = styled(AlertDialogPrimitive.Overlay, {
  backgroundColor: 'rgba(0, 0, 0, .15)',
  position: 'fixed',
  top: 0,
  right: 0,
  bottom: 0,
  left: 0,
});

const Content = styled(AlertDialogPrimitive.Content, {
  position: 'fixed',
  top: '50%',
  left: '50%',
  transform: 'translate(-50%, -50%)',
  width: 360,
  backgroundColor: 'white',
  borderRadius: 6,
  padding: 30,
});

const Title = styled(AlertDialogPrimitive.Title, {
  margin: '0 0 30px',
});

const Description = styled(AlertDialogPrimitive.Description, {
  margin: '0 0 30px',
  lineHeight: 1.5,
});

render(
  <AlertDialogPrimitive>
    <AlertDialogPrimitive.Trigger>Open</AlertDialogPrimitive.Trigger>
    <Overlay />
    <Content>
      <Title>Alert title</Title>
      <Description>
        This is an example of the alert description part. Are you sure?
      </Description>
      <AlertDialogPrimitive.Cancel as={Button} css={{ marginRight: 15 }}>
        Cancel
      </AlertDialogPrimitive.Cancel>
      <AlertDialogPrimitive.Action as={Button}>
        Accept
      </AlertDialogPrimitive.Action>
    </Content>
  </AlertDialogPrimitive>
);
```

#### AlertDialogCancel

`AlertDialogCancel` is a button that closes the `AlertDialog` and assumes no subsequent action is expected by the user. This button should be distinguished visually from `AlertDialogAction` buttons for the best usability.

#### AlertDialogAction

`AlertDialogAction` is a button that closes the `AlertDialog` and assumes that a subsequent action is expected by the user. These buttons should be distinguished visually from the `AlertDialogCancel` button for the best usability.

#### AlertDialogTitle

A component to provide an accessible name to be announced when an `AlertDialog` is opened. Alternatively, you can provide `aria-label` or `aria-labelledby` to `AlertDialogContent` and exclude this component.

#### AlertDialogDescription

A component to provide an accessible description to be announced when an `AlertDialog` is opened. Alternatively, you can provide `aria-describedby` to `AlertDialogContent` and exclude this component.
