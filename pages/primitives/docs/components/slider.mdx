---
title: Slider
name: slider
version: 0.0.1
description: An input where the user selects a value from within a given range.
features:
  - Can be controlled or uncontrolled.
  - Supports multiple thumbs.
  - Supports a minimum value between thumbs.
  - Supports touch or click on track to update value.
  - Supports Right to Left direction.
  - Full keyboard navigation.
---

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-slider
```

Then import the component in your component file.

```jsx
// Slider.tsx
import * as SliderPrimitive from '@radix-ui/react-slider';
```

### Basic example

Create your styled slider component from the slider primitives.

```jsx live manual
// Slider.tsx
// import * as SliderPrimitive from '@radix-ui/react-slider';

const Slider = styled(SliderPrimitive.Root, {
  position: 'relative',
  display: 'flex',
  alignItems: 'center',
  userSelect: 'none',
  touchAction: 'none',
  height: 16,
});

const SliderTrack = styled(SliderPrimitive.Track, {
  backgroundColor: 'gainsboro',
  position: 'relative',
  flexGrow: 1,
  height: 2,
});

const SliderRange = styled(SliderPrimitive.Range, {
  position: 'absolute',
  backgroundColor: 'dodgerblue',
  borderRadius: '9999px',
  height: '100%',
});

const SliderThumb = styled(SliderPrimitive.Thumb, {
  display: 'block',
  width: 16,
  height: 16,
  backgroundColor: 'white',
  border: '1px solid lightgray',
  borderRadius: '20px',
  ':focus': {
    outline: 'none',
    borderColor: 'dodgerblue',
  },
});

render(
  <Slider defaultValue={[50]}>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb />
  </Slider>
);
```

#### Disabled style

Use the availabe `data-*` attribute selectors to style the `disabled` state.

```jsx live manual compact
// Slider.tsx
// import * as SliderPrimitive from '@radix-ui/react-slider';

const Slider = styled(SliderPrimitive.Root, {
  position: 'relative',
  display: 'flex',
  alignItems: 'center',
  userSelect: 'none',
  touchAction: 'none',
  height: 16,
});

const SliderTrack = styled(SliderPrimitive.Track, {
  backgroundColor: 'gainsboro',
  position: 'relative',
  flexGrow: 1,
  height: 2,
});

const SliderRange = styled(SliderPrimitive.Range, {
  position: 'absolute',
  backgroundColor: 'dodgerblue',
  borderRadius: '9999px',
  height: '100%',
  [`${Slider}[data-disabled] &`]: {
    backgroundColor: 'darkgray',
  },
});

const SliderThumb = styled(SliderPrimitive.Thumb, {
  display: 'block',
  width: 16,
  height: 16,
  backgroundColor: 'white',
  border: '1px solid lightgray',
  borderRadius: '20px',
  outline: 'none',
  [`${Slider}:not([data-disabled]) &:focus`]: {
    borderColor: 'dodgerblue',
  },
});

render(
  <Slider defaultValue={[50]} disabled>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb />
  </Slider>
);
```

#### SliderTrack

`SliderTrack` is the track part.

#### Range

`SliderRange` is the range part. Must live inside `SliderTrack`.

#### Thumb

`SliderThumb` is the thumb part. You can render multiple thumbs.
