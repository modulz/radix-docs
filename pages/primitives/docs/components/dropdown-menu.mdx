---
title: Dropdown Menu
name: dropdown-menu
version: 0.0.1-1
description: Displays a menu to the user—such as a set of actions or functions—triggered by a button.
aria: https://www.w3.org/TR/wai-aria-practices/#menubutton
features:
  - Can be controlled or uncontrolled.
  - Supports items, labels, groups of items.
  - Supports checkable items (single or multiple).
  - Customise side, alignment, offsets, collision handling.
  - Optionally render a pointing arrow.
  - Focus is fully managed.
  - Full keyboard navigation.
  - Dismissing and layering behavior is highly customisable.
---

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-dropdown-menu
```

### Anatomy

Import the components and piece the parts together.

```jsx
import {
  ContextMenu,
  Trigger,
  Content,
  Item,
  Group,
  Label,
  CheckboxItem,
  RadioGroup,
  RadioItem,
  ItemIndicator,
  Separator,
  Arrow,
} from '@radix-ui/react-dropdown-menu';

export default () => (
  <ContextMenu>
    <Trigger />
    <Content>
      <Item />
      <Separator />
      <CheckboxItem />
      <RadioGroup>
        <RadioItem />
      </RadioGroup>
      <Item>
        <ItemIndicator />
      </Item>
    </Content>
  </ContextMenu>
);
```

### Basic example

Create your styled dropdown menu component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as DropdownMenuPrimitive from '@radix-ui/react-dropdown-menu';

const Content = styled(DropdownMenuPrimitive.Content, {
  minWidth: 130,
  backgroundColor: 'white',
  borderRadius: 6,
  padding: 5,
  boxShadow: '0px 5px 15px -5px hsla(206,22%,7%,.15)',
});

const Item = styled(DropdownMenuPrimitive.Item, {
  fontSize: 13,
  padding: '5px 10px',
  borderRadius: 3,
  cursor: 'default',

  '&:focus': {
    outline: 'none',
    backgroundColor: 'dodgerblue',
    color: 'white',
  },
});

const Arrow = styled(DropdownMenuPrimitive.Arrow, {
  fill: 'white',
});

render(
  <DropdownMenuPrimitive.Root>
    <DropdownMenuPrimitive.Trigger>
      Trigger
    </DropdownMenuPrimitive.Trigger>
    <Content>
      <Item>Undo</Item>
      <Item>Redo</Item>
      <Item>Cut</Item>
      <Item>Copy</Item>
      <Item>Paste</Item>
      <Arrow />
    </Content>
  </DropdownMenuPrimitive.Root>
);
```

### Accessibility

Adheres to the [Menu Button WAI-ARIA design pattern](https://www.w3.org/TR/wai-aria-practices/#menubutton) and uses [roving tabindex](https://www.w3.org/TR/wai-aria-practices/examples/menu-button/menu-button-actions.html) to manage focus movement among menu items.

#### Trigger

The button that triggers the `DropdownMenu`. It is also used as the anchor the `DropdownMenuContent` is positioned against.
